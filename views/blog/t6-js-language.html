Javascript Language to my non tech friend
Describe the use of :

Parentheses ()

The parenthesis are used a lot. They can follow a function (empty or not), they contain the conditions of if, for and while.
When folloing a function, they help targetting the subject on which the function will take action.
On other situations, they will help structure the content of the code, in a very similar way to mathematics.

Brackets []

Brackets are used for declaration of arrays. The look of the punctionation used define the type of variable created. Defining an array over an object is favorable when we need to stock a list of values or strings.

Curly Braces {}

Curly braces are used in Javascript for two different purpose. They are surrounding the content of a function, if, for and while statement and they define an object.
The first use of curly brace is looking more like actual puntuation to me. They are used to structure the different conditions and give them some scope (so they don't get mixed into another). They make the content cleaner and more readable by both human and machine.
When curly braces are used for object definition, it ressembles more to a grammatical convention. Objects are declared with curly braces (may they be empty or with content),so that is what should be used in this scenario. Defining an object is different to an array as an object will contain both properties and their respectives values.

Single quotes '' and Double quotes ""

Single quotes and double quotes seems to act as the same to me, without being interchangeable (you can't expect to start a statement with a single quote and end it with a double quote).
They are used in string declaration. Therefore, a choice has to be made for the one that will be needed the least in the language used in the string content.
It seems to be that javascript user are mostly using double quote to declare strings. I personnaly have been using single quotes out of ease (only have to use one finger to type it on the keyboard - but noticing that the norm seems to tend to go towards double quotes, I will match it).
