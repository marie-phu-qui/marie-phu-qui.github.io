What is a scope and how it works in JavaScript

The concept of scope

The scope is the area in which something can be recognized as what it is. The scope can be global or local. If global the object will be defined and recognized as the same value for the whole file; If local it will have to be redeclared every time we need it again outside its scope - Otherwise it will be "Undefined".

In the Javascript language

The scope takes action on all Javascript objects. Function define a scope. Variables declared with the term "let" will be regnonizable on the local scope, and the ones declared under "const" will be declared on the global one.
A local scope will be defined by an open curly bracket and will stop and the corresponding closing curly braket.
